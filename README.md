# DrawingJS
Uses HTML5 Canvas and JavaScript to draw shapes.

Currently includes:
<ul>
<li>Rectangles</li>
<li>Circles</li>
<li>Text</li>
<li>Images</li>
<li>Lines</li>
<li>Canvas width and height variables</li>
</ul>
Coming soon:
<ul>
<li>Key Pressed & Released Functions</li>
<li>Mouse Pressed & Released Functions</li>
<li>Ellipses/Ovals</li>
</ul>
